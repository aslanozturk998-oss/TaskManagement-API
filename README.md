# Task Management API

A scalable and robust RESTful API designed for managing tasks and categories. Built with **ASP.NET Core**, this project demonstrates modern software engineering practices including multi-tier architecture concepts, efficient data mapping, and automated testing.

## 🚀 Key Features
* **Architecture:** Structured with separation of concerns in mind, utilizing Controllers, Models, and DTOs.
* **Data Access:** Entity Framework Core (Code-First approach) with relational database management.
* **Error Handling:** Custom Global Exception Handling Middleware for standardized, client-friendly error responses.
* **Quality Assurance:** Unit tests implemented using **xUnit** and EF Core In-Memory database to ensure business logic integrity.

## 💻 Tech Stack
* **Framework:** .NET 10.0 (C#)
* **Database:** SQL Server & Entity Framework Core
* **Testing:** xUnit
* **API Documentation:** Swagger / OpenAPI
